# COOL.FI

Prototype social-first event feed for Firenze.

The current build is a single static `index.html` that can be opened directly in a browser or hosted with GitHub Pages.

## Roadmap

- Move events out of inline JavaScript into a data source.
- Add a daily AI agent that collects official events, verifies sources, and updates the feed.
- Use official media and social links where available.
- Add review workflow before publishing low-confidence updates.
